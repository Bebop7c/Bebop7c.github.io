About

This site highlights notable university and personal projects undertaken throughout my career. It is a technical CV focusing on practical work and accomplishments.

Schmidt Telescope Project

My final year university project focuses on building and controlling a Schmidt telescope. The project involves optical alignment, embedded control, and software for data acquisition.

DIY VESC Onewheel

A personal endeavor to design and build a single-wheel electric vehicle using an open source VESC controller. This project encompasses mechanical design, motor control, and battery management.

Long Distance Running Computer Vision Form Analysis

An academic project applying computer vision techniques to analyze running form over long distances. The codebase implements pose detection and generates feedback to improve efficiency.

Computer Vision Leather Hide Error Detection

At work I developed a system that detects blemishes in leather hides using image processing and machine learning. The tool speeds up quality assurance by automatically locating defects.

Embedded Systems Wacky Racers Robot

In this university module, I built a small autonomous robot for a racing challenge. The focus was on embedded programming, sensor integration, and real-time decision making.

Robocup Competition

My team placed third in a Robocup event after creating a robot that could navigate and complete tasks autonomously. Responsibilities included mapping and control algorithms.
